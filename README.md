#  Nihal Butik - Stok ve Rezarvasyon Sistemi

Java FX ile geliştirdiğim butik yönetim sistemi. Abiye kiralama, satış ve stok takibi.

## Özellikler
-Ürün ve kategori yönetimi
-Müşteri takibi
-Rezervasyon sistemi
-SQLite veritabanı
-Java FX arayüz

##Teknolojiler
-Java 17
-Java FX
-SQLite
-Maven
